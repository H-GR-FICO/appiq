Část A
Logotypy — horizontální lockup
Hlavní logo platformy pro nav header, titulní stranu dokumentu a prezentace. Všechny varianty používají autentické HOPI vektorové paths.
L1 — Horizontal Výchozí
Horizontal — Výchozí (aktuálně nasazeno)
HOPI brand vektory + svislá linka + "App" tlumená šedá + "IQ" zelená. Jemný kontrast, moderní a čistý. Aktuálně použito v nav headeru a cover stránce.
nav header · cover strana · promo web
L2 — Bold AppIQ
Horizontal — Bold AppIQ
"App" plná bílá tučná + "IQ" světlá zelená (#00cc55). Rovnoměrnější vizuální váha HOPI a AppIQ — assertivnější dojem, silnější brand recall AppIQ.
nav header · cover strana · print
L3 — Stacked Badge
Stacked Badge
HOPI brand path nad "AppIQ" — dva řádky v badge formátu. Kompaktní šířka, silná výška. Jedinečné a snadno zapamatovatelné. Ideální pro titulní strany a viditelné umístění.
cover strana · titulní badge · print · SharePoint icon
L4 — IQ Mark + Text
IQ Mark + Text
Zelený IQ čtverec jako standalone "app icon" + HOPI AppIQ text vpravo. Modulární systém — IQ mark lze použít samostatně jako favicon/badge, text vedle pro header.
web header · favicon + wordmark combo · Teams app tile
Část B
Piktogramy — icon / favicon formát
Standalone symbol platformy. Použití: favicon (16–512px), Teams badge, SharePoint site icon, nadpis prezentace, vodoznak.
B1 — Hexagon IQ
Skvělý v 16 × 16 i 512 × 512
Hexagon IQ
Hexagon = tech/enterprise standard (viz AWS, SAP, Microsoft). Čistý, okamžitě čitelný v jakémkoli rozlišení. Nejlepší volba pro favicon.
B2 — App Grid
Launcher · Teams tile · tiles view
App Grid Launcher
Mřížka appek s IQ uprostřed — odkazuje přímo na "aplikační platformu". Vhodné jako Teams/M365 launcher ikona.
B3 — Neural Orbit
AI messaging · platform hub
Neural Orbit
AI vizuální jazyk — uzly propojené kolem IQ středu. Silný tech/AI dojem. Při malých rozlišeních méně čitelný — ideální 48px+.
B4 — Circuit H
HOPI identity · digital DNA
Circuit H
Písmeno H s circuit-board elementy — spojuje HOPI identitu s tech/digitálním DNA. Výrazný, dobře čitelný. Odkazuje explicitně na HOPI.
B5 — Shield IQ
Enterprise · security · corporate
Shield IQ
Štít = enterprise, bezpečnost, spolehlivost — tradiční korporátní symbol. Silné poselství důvěry. Vhodné pokud cílíte na compliance/governance messaging.
B6 — Diamond IQ
Premium · quality · modern
Diamond IQ
Rotovaný čtverec — prémiový, geometrický, dynamický. Dobře funguje jako velký badge. Při malých rozlišeních (favicon 16px) méně výrazný tvar.
Doporučená kombinace
+
L1 Horizontal + B1 Hexagon IQ
Nav header: logotyp L1  ·  Favicon / badge: hexagon B1
Kombinace L1 + B1 zachovává autentický HOPI brand v logotypu a přidává silný, škálovatelný hexagon jako piktogram. Hexagon funguje výborně jako favicon (16px), Teams app tile (32px), SharePoint site icon (96px) i jako dekorativní badge v prezentaci. Pro titulní strany doporučuji alternativně L3 (Stacked Badge) — silnější vizuální dojem na velké ploše. Napište svou volbu ("L_ + B_") a ihned ji zabudujeme.
