— 5 ZÁKONŮ PLATFORMY

1
Single Responsibility
Každý modul dělá JEDNU věc. Nikdy víc.
2
Event Bus
Moduly komunikují POUZE přes AppIQ.emit/on. Žádné přímé volání.
3
Data Layer
Veškerý přístup k datům jde přes _data.js. Nikdy fetch() přímo v HTML.
4
DRY
Stejný kód na dvou místech patří do modulu. Žádná duplicita.
5
Backend-Ready
Swap JSON → REST API → SQL = změna POUZE v _data.js.
